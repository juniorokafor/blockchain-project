Blockchain Ticketing DApp

A decentralised ticketing system built on the Ethereum Sepolia testnet, using an ERC-20 smart contract to issue tickets as tokens.

Features

Create Wallet Generate a new encrypted wallet (V3 keystore) Check Balance Look up SETH and ticket balance for any address Buy Ticket Purchase a ticket token using SETH Return Ticket Transfer a ticket back to the vendor for a refund

How to use

  1. Create a wallet (with a password for encryption) and download the keystore JSON.
  2. Get free SETH from a Sepolia faucet (e.g. sepoliafaucet.com).
  3. Buy a ticket — load your keystore + password, then send SETH to the contract.
  4. Verify or return your ticket — the doorman can check your balance, or you can return it back to the vendor.

Network
